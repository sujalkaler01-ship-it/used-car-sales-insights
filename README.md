# used-car-sales-insights

A beginner-friendly Python data visualization project that analyzes used car sales data and shows useful insights through simple charts.

## Project Overview

This project explores a used car sales dataset to understand patterns in car prices, brands, fuel types, transmission types, and other important features. The main goal is to use basic Python libraries to clean, analyze, and visualize the data in an easy-to-understand way.

## Problem Statement

Used car buyers and sellers often want to understand what affects the price of a car. This project helps analyze used car sales data to find trends and insights that can support better decision-making.

## Objectives

- Load and understand the used car dataset
- Clean and prepare the data for analysis
- Analyze car price trends
- Compare different car brands
- Visualize fuel type and transmission distribution
- Create simple and clear charts for insights

## Tools and Libraries Used

- Python
- Google Colab / Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## Key Features

- Data loading and basic exploration
- Simple data cleaning
- Bar charts for comparison
- Count plots for categorical data
- Price analysis using visualizations
- Beginner-friendly code and explanations

## Visualizations Included

- Top car brands by count
- Fuel type distribution
- Transmission type distribution
- Car price comparison
- Year-wise or mileage-based price insights

## Dataset

The project uses a used car sales dataset containing details such as:

- Car brand/name
- Selling price
- Year
- Fuel type
- Seller type
- Transmission
- Owner type
- Kilometers driven
